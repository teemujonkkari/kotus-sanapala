# kotus-sanapala

A simple cheat sheet cli app for Finnish words for the game Sanapala.

## Requirements

- Terminal
- Node.js v16.0.0 or higher
- yarn v1.22.10 or higher

## Installation

```bash
yarn install && chmod +x app.js
```

## Usage

```bash
./app.js --letters=akssi --limit=5
```

## Results

```bash
┌─────────┬─────────┐
│ (index) │ Values  │

```
